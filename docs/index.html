Cross-Platform Explorer

A fast, tiny desktop file explorer for Windows, macOS, and Linux. One-click install, automatic updates, built with Tauri + Svelte.

⬇ Download latest View on GitHub

Windows · macOS · Linux — installers built automatically for every release.

Why it's different

🪶

Tiny footprint

Native binaries in the single-digit megabytes — no bundled browser engine.

🔄

Auto-updates

Cryptographically signed updates install themselves on the next launch.

🖱️

One-click install

Proper native installers: .msi/.exe, .dmg, and .AppImage/.deb.

🤖

AI-maintainable

Documented architecture and build pipeline so an assistant can extend it safely.

Build it yourself

git clone https://github.com/StewartScottRogers/cross-platform-explorer.git
cd cross-platform-explorer
npm install
npm run tauri dev
