Cross-Platform Explorer

A fast, tiny file explorer for Windows, macOS, and Linux — with a live view of what your AI coding agents are doing to your files.

⬇ Download Get started

Windows · macOS · Linux — all downloads & versions

Get up and working in a minute

No account, no setup. Download, open a folder, and go.

1

Install

Grab the installer for your OS above and run it — a proper native installer (.msi/.exe, .dmg, .AppImage/.deb). It updates itself from then on.

2

Browse

It opens at your home folder. Double-click to dive in, use the breadcrumb or ↑ / ← / → to move around, and open tabs or a second pane for side-by-side work.

3

Do more

Search and filter, batch-rename, tag, extract archives, see a disk-usage treemap, and press F1 anywhere for the built-in docs. Every section has its own help button.

Everything you expect from a file manager

Fast, small, and predictable — the plain explorer stays that way whether or not you use the AI features.

🗂️

Browse & navigate

Tabs, an optional dual-pane commander, breadcrumb path bar, full keyboard navigation, and back / forward / up history.

🔎

Find & organize

Content search, filename search with glob patterns, a duplicates finder, saved smart folders, tags, favorites, and batch rename.

📦

Files & archives

Copy, move, cut/paste, duplicate, trash, new file/folder, plus zip compress and extract — and drag files in and out of the OS.

👁️

Preview & insight

Inline previews for images, PDF, markdown, code, audio & video, universal thumbnails, a disk-usage Space analyzer, and a compare studio.

🎨

Fast & native

Light/dark themes, instant streaming of large folders, a diagnostics mode, and a built-in contextual documentation library.

🔄

Installs & updates itself

Small native binaries (single-digit MB, no bundled browser), proper OS installers, and cryptographically signed auto-updates.

what makes it different

See what your AI agents are doing

Coding agents touch your filesystem constantly. This explorer makes that visible — and lets you run them right where your files live.

🛰️

Agent Watch

A live view in the left pane of an AI coding agent's filesystem activity — the files it reads and the edits it makes, as it works, with diff peeks.

🤖

Agent Deck

Launch and manage agentic coding CLIs (Claude Code, aider, …) as sandboxed sidecars against any folder — pick any provider and model.

🧩

Grid, Board & Swarms

Tile every running agent in a grid, track work on a ticket board, or run a coordinated multi-agent swarm on one project.

🔗

Repositories & Workbench

Browse and clone GitHub and other forges in-app (host-brokered, allow-listed), and review the working tree's git diff in the Workbench.

The AI features are opt-in — with them switched off, the plain explorer stays fast, small, and predictable.

Building on it?

It's a Tauri v2 app — Rust backend, Svelte + TypeScript frontend. The README has everything a developer needs: build, run, architecture, updater setup, and releasing.

git clone https://github.com/StewartScottRogers/cross-platform-explorer.git
cd cross-platform-explorer
npm install
npm run tauri dev
