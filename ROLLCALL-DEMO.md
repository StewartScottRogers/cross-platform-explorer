- **package-lock.json** — the locksmith, ensuring everyone gets exactly the right dependency versions

- **PURPOSE.md** — the philosopher, explaining why this app exists and what it stands for

- **vite.config.ts** — the build orchestrator, configuring Vite's dev server and bundling

That's the team! 24 files working together to make this cross-platform file explorer happen.
