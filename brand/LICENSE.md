## Provenance — why the rights are clean

Every asset in this `brand/` folder (`icon.svg`, `logo.svg`, `mark-mono.svg`, and any raster exports
derived from them) was **authored from scratch specifically for Cross-Platform Explorer**. Each is
built **solely from primitive vector geometry** — SVG paths, rectangles, circles, and CSS-color
gradients — that were composed by hand for this project.

These assets contain **no third-party intellectual property**:

- **No** stock art, clip-art, icon-library glyphs, or downloaded images.
- **No** traced, copied, or derivative artwork from any existing logo or icon.
- **No** embedded, subsetted, or redistributed font files. The wordmark is rendered as live text in a
  *generic system sans* stack (Segoe UI / system-ui / Arial); the font itself is **not** included or
  distributed. For any exported/print artwork, convert that text to vector **outlines** so the final
  file carries no font dependency at all.
- **No** AI-generated raster imagery from third-party services; the vectors are deterministic,
  human-authored markup you can read in full in each `.svg`.

Because the marks are 100% original and dependency-free, the project holds **clear, unencumbered
rights** to reproduce, modify, distribute, and **register them as trademarks**.
