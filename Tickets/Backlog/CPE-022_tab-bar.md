## Summary

Windows 11 Explorer has a tab strip: each tab holds its own location and history, with a "+" to add
and an "x" to close.

## Acceptance Criteria

- [ ] Tab strip renders across the top with the active tab highlighted
- [ ] "+" opens a new tab at Home; "x" closes a tab (last tab cannot be closed)
- [ ] Each tab keeps its own current path and back/forward history
- [ ] Switching tabs restores that tab's location without a reload flash
- [ ] Tab shows the current folder name and a matching icon
