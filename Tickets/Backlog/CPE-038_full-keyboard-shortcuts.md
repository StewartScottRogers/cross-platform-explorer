## Summary

Implement the standard Explorer shortcuts we are missing, so muscle memory transfers.

## Acceptance Criteria

- [ ] Ctrl+T new tab, Ctrl+W close tab, Ctrl+Tab / Ctrl+Shift+Tab switch tabs
- [ ] Ctrl+A select all, Ctrl+F focus search, F5 refresh, Alt+Up go up
- [ ] Ctrl+Shift+N new folder, F2 rename, Del / Shift+Del delete
- [ ] Ctrl+C/X/V clipboard, Alt+Enter properties, Alt+P toggle details pane
- [ ] Shortcuts are ignored while an inline editor or text field has focus
- [ ] A shortcut cheatsheet is documented in the README
