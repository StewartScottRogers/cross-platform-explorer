## Summary

human-readable type names ("PNG image", "Markdown file") for the Type column.

## Acceptance Criteria

- [ ] Extension -> icon mapping for common types (image, doc, sheet, code, archive, audio, video, pdf)
- [ ] Extension -> human-readable type name for the Type column

- [ ] Mapping is a pure module with unit tests
