# Cross-Platform Explorer

Cross-Platform Explorer is a fast, small, predictable desktop file explorer for Windows, macOS, and
Linux. It does the everyday things well — browse folders, preview and edit files, search, manage tabs —
and layers optional **AI Agent Workspace** modes on top for developers who run coding agents.

## Two halves

- **The explorer** — the core file manager. It is deliberately lean: with every optional mode switched
  off, it stays fast to launch, light on memory, and behaves the way you expect a file explorer to.
- **The Agent Workspace** — optional, additive views for running and coordinating AI coding agents:
  the **Agent Deck**, the **Agent Grid**, the **Agent Board**, **Swarms** (several agents on one task),
  the **Workbench**, and shared agent memory. These appear as entries in the left sidebar and the
  Application menu.

## Where things live

- **Left sidebar** — Home, Favorites, drives, the running **Agents**, and the workspace entries
  (Repositories, Agent Board, Workbench).
- **Main area** — the file list (or Home), with tabs across the top and a status bar along the bottom.
- **Preview pane** — shows the selected file (text, images, markdown, PDF, and more), with in-place edit
  for text.
